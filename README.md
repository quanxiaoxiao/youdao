## Install

```
npm install
npm run build
sudo mv ./youdao.js /usr/local/bin/youdao
sudo chmod +x /usr/local/bin/youdao
```

## Usage

English To Chinese

```
$youdao test
##################################
# n. 试验；检验
# vt. 试验；测试
# vi. 试验；测试
# n. (Test)人名；(英)特斯特
##################################
```

Chinese To English
```
$youdao 测试
##################################
# [试验] test
# measurement
##################################
```
